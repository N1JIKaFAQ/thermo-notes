第八章 液体

李椿·章立源·钱尚武《热学》 — 期末补热学笔记

概述 p291

液体介于气体和固体之间：既有流动性（像气体），又有一定体积（像固体）。液体的微观结构特点是近程有序、远程无序。

§1 液体的微观结构 p291-296

  • 近程有序：每个分子附近有规则排列（几个分子间距范围）
  • 远程无序：超出近程范围后排列不规则
  • 分子在平衡位置附近振动，但可发生跳跃式移动
  • 分子间作用力较强，分子间距接近固体

§2 液体中的输运过程 p296-301

液体的粘度 p296-299

液体粘滞系数 $\eta$ 远大于气体。温度升高时液体粘度减小（与气体相反）。

$$\eta = \eta_0 e^{E/(kT)}$$

$E$ — 分子流动活化能

泊肃叶公式 p300-301

液体在圆管中层流的流量：

$$Q = \frac{\pi R^4}{8\eta}\frac{\Delta p}{l}$$

$R$ — 管半径；$\Delta p$ — 两端压强差；$l$ — 管长；$\eta$ — 粘度

§3 液体的表面性质 p301-316

一、表面张力 p301-305

★ 表面张力 (8.7)
$$f = \alpha L$$

$\alpha$ — 表面张力系数（N/m），方向与液面相切，垂直于线段指向液面收缩方向。

$\alpha$ 随温度升高而减小，随液体纯度而变化。

二、表面能 p305-307

★ 表面能的定义 (8.8)
$$\alpha = \frac{\Delta E}{\Delta S}$$

表面张力系数等于单位表面面积的自由能（J/m²）。

外力做功：$\Delta A = \alpha \Delta S$

三、弯曲液面的附加压强 p307-311

★ 球形液面附加压强 (拉普拉斯公式)
$$\Delta p = \frac{2\alpha}{R}$$

对肥皂泡（有两个表面）：$\Delta p = 4\alpha/R$

对任意弯曲液面：$\Delta p = \alpha\left(\frac{1}{R_1} + \frac{1}{R_2}\right)$

  • 凸液面：$\Delta p > 0$（附加压强指向液体内部）
  • 凹液面：$\Delta p < 0$（附加压强指向液体外部）

四、毛细现象 p311-316

★ 毛细管中液面高度
$$h = \frac{2\alpha\cos\theta}{\rho g r}$$

$r$ — 毛细管半径；$\theta$ — 接触角（润湿角）；$\rho$ — 液体密度

  • $\theta < 90^\circ$（润湿）：$h > 0$，液面上升
  • $\theta > 90^\circ$（不润湿）：$h < 0$，液面下降（如汞）
  • $\theta = 0^\circ$：完全润湿，$h = 2\alpha/(\rho g r)$

接触角与润湿 p314

$$\cos\theta = \frac{\alpha_{SG} - \alpha_{SL}}{\alpha_{LG}}$$

$\alpha_{SG}, \alpha_{SL}, \alpha_{LG}$ 分别为固-气、固-液、液-气界面张力

§4 液体的蒸发与沸腾 p316-322

蒸发 p316-318

  • 发生在液体表面的汽化过程，任何温度下均可进行
  • 蒸发使液体降温（动能大的分子先逃逸）
  • 影响因素：表面积、温度、通风条件

沸腾 p318-322

  • 整个液体内部产生气泡的剧烈汽化
  • 沸腾温度（沸点）与压强有关
  • 沸腾条件：饱和蒸气压 = 外界压强

本章核心公式速查

  1. 表面张力：$f = \alpha L$ p303
  2. 表面能：$\alpha = \Delta E/\Delta S$ p306
  3. 球形液面附加压强：$\Delta p = 2\alpha/R$ p309
  4. 拉普拉斯公式：$\Delta p = \alpha(1/R_1 + 1/R_2)$ p310
  5. 毛细管高度：$h = 2\alpha\cos\theta/(\rho g r)$ p313
  6. 接触角：$\cos\theta = (\alpha_{SG} - \alpha_{SL})/\alpha_{LG}$ p314
  7. 泊肃叶公式：$Q = \pi R^4\Delta p/(8\eta l)$ p301
  8. 液体粘度：$\eta = \eta_0 e^{E/(kT)}$ p298
