第二章 气体分子运动论的基本概念

李椿·章立源·钱尚武《热学》 — 期末补热学笔记

本章概述 p45

从分子运动论的观点阐明气体的一些宏观性质和规律。建立理想气体的微观模型，阐明压强和温度的实质，推证基本气体定律。

§1 物质的微观模型 p45-48

三个基本出发点

  1. 宏观物体由大量分子（原子）组成，分子间有空隙
  2. 分子永不停息地做无规则运动，剧烈程度与温度有关
    • 扩散现象（气体、液体、固体）
    • 布朗运动：悬浮颗粒受分子撞击不平衡所致，间接证明分子热运动
  3. 分子之间有相互作用力（引力和斥力）
    • 引力：使固体液体保持一定体积
    • 斥力：使固体液体难以压缩

三态微观解释 p49

  • 固体：分子在平衡位置附近振动（温度低，分子力占主导）
  • 液体：分子无固定平衡位置，但尚未分散远离
  • 气体：分子分散远离，近似自由运动（温度高，热运动占主导）

§2 理想气体的压强 p49-53

一、理想气体的微观模型 p50

  1. 分子本身线度 远小于 分子间平均距离（可忽略）
  2. 除碰撞瞬间外，分子间及分子与器壁间 无相互作用
  3. 分子间及分子与器壁的碰撞是 完全弹性的（动能不损失）

还可以忽略分子所受的重力。

二、压强公式的推导 p51-53

核心思路：大量分子不断碰撞器壁 → 统计平均 = 持续压力

设容器体积 V，总分子数 N，分子数密度 n = N/V，分子质量 m。

Step 1：一个分子碰壁的冲量 = $2mv_{ix}$（完全弹性，x方向速度反向）

Step 2：时间 dt 内能与 dA 相碰的分子数 = $n_i v_{ix} dt dA$

Step 3：总冲量求和，取统计平均

★ 压强公式推导结果
$$p = nm\overline{v_x^2}$$ $$\overline{v_x^2} = \overline{v_y^2} = \overline{v_z^2} = \frac13\overline{v^2}$$ $$\boxed{p = \frac13 nm\overline{v^2} = \frac23 n\overline{\varepsilon}}$$

其中 $\overline{\varepsilon} = \frac12 m\overline{v^2}$ 为分子平均平动动能

意义：宏观量 p 与微观量 $\overline{\varepsilon}$ 联系起来。p是统计平均量，不是力学量。

§3 温度的微观解释 p54-58

一、温度与平均平动动能 p54-55

联立压强公式 $p = \frac23 n\overline{\varepsilon}$ 和状态方程 $pV = \frac{M}{\mu}RT$：

$$\overline{\varepsilon} = \frac{3}{2} \cdot \frac{R}{N_A} T = \frac32 kT$$

其中：

  • $N_A = 6.022 \times 10^{23}\; \text{mol}^{-1}$ — 阿伏伽德罗常数
  • $k = \dfrac{R}{N_A} = 1.38 \times 10^{-23}\; \text{J/K}$ — 玻尔兹曼常数

结论：气体分子的平均平动动能 只与温度有关，与热力学温度成正比。

→ 温度是大量分子热运动的集体表现，是统计概念。
对单个分子谈论温度没有意义。

方均根速率 p57

★ 方均根速率
$$\sqrt{\overline{v^2}} = \sqrt{\frac{3kT}{m}} = \sqrt{\frac{3RT}{\mu}}$$

$\sqrt{\overline{v^2}} = \sqrt{\dfrac{3 \times 8.31 \times 273}{2.02 \times 10^{-3}}} = 1838 \; \text{m/s}$

二、对理想气体定律的推证 p58-59

由 $p = \frac23 n\overline{\varepsilon}$ 和 $\overline{\varepsilon} = \frac32 kT$ 可得：

$$p = nkT \qquad\text{(2.8)}$$

1. 阿伏伽德罗定律

$p = nkT$ → 同温同压下，相同体积内分子数相等。

STP 下：$n = \dfrac{p}{kT} = 2.69 \times 10^{25}\; \text{m}^{-3}$（洛喜密脱数）

2. 道耳顿分压定律

混合气体中，各种气体温度相同 → $\overline{\varepsilon}_i$ 相同

$$p = \frac23(n_1+n_2+\cdots)\overline{\varepsilon} = p_1+p_2+\cdots$$

§4 分子力 p59-62

分子力公式（半经验）

$$f = \frac{\lambda}{r^s} - \frac{\mu}{r^t}, \quad (s > t)$$

第一项（正）= 斥力，第二项（负）= 引力。

  • $r = r_0$：平衡位置，$f=0$
  • $r < r_0$：强大斥力（有效作用距离比引力小）
  • $r > r_0$：引力
  • 分子有效直径 ≈ $10^{-10}$ m

分子势能曲线

$$E_P = \frac{\lambda'}{r^{s-1}} - \frac{\mu'}{r^{t-1}}$$

$r = r_0$ 处势能最小。从远处来的分子动能先增大（引力做功），到 $r < r_0$ 后急剧减速并反弹 → 弹性碰撞的微观机制。

简化模型

模型特点
刚球模型$r < d$ 时 $E_P = \infty$，$r > d$ 时 $E_P = 0$
苏则朗模型$r < d$ 时 $E_P = \infty$，$r > d$ 时有引力

§5 范德瓦耳斯气体的压强 p63-66

范德瓦耳斯方程的推导

出发点：理想气体忽略了分子体积（斥力）和分子间引力。

一、体积修正（斥力）p63-64

分子看作刚球 → 自由活动空间 = $v - b$

$$p = \frac{RT}{v - b}$$

$b$ ≈ 一摩尔气体所有分子体积总和的 4 倍

$b = 4N_A \cdot \dfrac43\pi\left(\dfrac{d}{2}\right)^3 \approx 10^{-5}\;\text{m}^3$（STP下仅为 v 的万分之四）

二、引力修正 p65-66

靠近器壁的分子受内部气体的吸引力 → 碰壁动量减小 → 压强减小

$$\Delta p \propto n^2 \propto \frac{1}{v^2} \quad\rightarrow\quad \Delta p = \frac{a}{v^2}$$

最终结果：范德瓦耳斯方程

$$\left(p + \frac{a}{v^2}\right)(v - b) = RT \qquad\text{(对1摩尔气体)}$$

$a$：引力修正系数，反映分子间吸引力大小（查表）
$b$：体积修正系数，反映分子本身体积

本章核心公式速查

  1. 压强公式：$p = \frac13 nm\overline{v^2} = \frac23 n\overline{\varepsilon}$ p53
  2. 平均平动动能：$\overline{\varepsilon} = \frac32 kT$ p55
  3. 玻尔兹曼常数：$k = 1.38 \times 10^{-23}\; \text{J/K}$
  4. 方均根速率：$\sqrt{\overline{v^2}} = \sqrt{3kT/m} = \sqrt{3RT/\mu}$ p57
  5. p = nkT p58
  6. 范德瓦耳斯方程：$(p + a/v^2)(v - b) = RT$ p66
