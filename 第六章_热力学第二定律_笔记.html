第六章 热力学第二定律

李椿·章立源·钱尚武《热学》 — 期末补热学笔记

概述 p201

热力学第一定律说明能量守恒，但并未指出过程的方向性。热力学第二定律独立于第一定律，揭示了自然界中一切与热现象有关的宏观过程的方向性和不可逆性。

§1 自然过程的方向性 p201-206

不可逆过程的实例

  • 热传导：热量自发地从高温传向低温，反之不成立
  • 功变热：机械功可以通过摩擦全部转化为热，反之不可
  • 扩散：气体自发混合，不会自发分离
  • 自由膨胀：气体向真空膨胀，不会自动缩回

结论：一切与热现象有关的宏观过程都是不可逆的。

§2 热力学第二定律的表述 p206-213

★ 开尔文-普朗克表述

不可能从单一热源吸取热量，使之完全变为有用功而不产生其他影响。

→ 第二种永动机（从单一热源吸热全部做功）不可能实现。

★ 克劳修斯表述

热量不能自发地从低温物体传到高温物体而不引起其他变化。

→ 热传导的不可逆性。制冷机需外界做功才能实现逆向传热。

两种表述的等价性：可以证明开尔文表述和克劳修斯表述是等价的。违反其中之一必然导致违反另一个。

热机与制冷机 p209-213

热机效率
$$\eta = \frac{A}{Q_1} = 1 - \frac{Q_2}{Q_1}$$
制冷系数
$$e = \frac{Q_2}{A} = \frac{Q_2}{Q_1 - Q_2}$$

$Q_1$ — 从高温热源吸热；$Q_2$ — 向低温热源放热；$A$ — 对外做功

§3 卡诺定理 p213-220

★ 卡诺定理
  1. 在相同的高温热源 $T_1$ 和低温热源 $T_2$ 之间工作的一切可逆热机，其效率都相等，与工作物质无关
  2. 一切不可逆热机的效率都小于可逆热机的效率

卡诺热机效率

$$\eta_C = 1 - \frac{T_2}{T_1}$$

只取决于两个热源的温度，与工作物质种类无关。

重要推论：

  • 可逆热机效率最大
  • $\displaystyle \frac{Q_1}{T_1} = \frac{Q_2}{T_2}$（可逆卡诺循环）
  • $\displaystyle \frac{Q_1}{T_1} \leq \frac{Q_2}{T_2}$（不可逆）
  • $\displaystyle \oint \frac{\delta Q}{T} \leq 0$（克劳修斯不等式）

§4 熵 p220-228

熵的定义 p220-222

熵是状态函数，其微分定义为：

★ 克劳修斯熵公式
$$dS = \frac{\delta Q_{\text{rev}}}{T}$$ $$\Delta S = S_2 - S_1 = \int_1^2 \frac{\delta Q_{\text{rev}}}{T}$$

熵是广延量，单位 J/K。

理想气体的熵变计算 p222-226

$$\Delta S = nC_V\ln\frac{T_2}{T_1} + nR\ln\frac{V_2}{V_1}$$ $$\Delta S = nC_P\ln\frac{T_2}{T_1} - nR\ln\frac{p_2}{p_1}$$

等温过程（$T=\text{const}$）：$\Delta S = nR\ln(V_2/V_1)$

等容过程（$V=\text{const}$）：$\Delta S = nC_V\ln(T_2/T_1)$

等压过程（$p=\text{const}$）：$\Delta S = nC_P\ln(T_2/T_1)$

相变过程的熵变 p226

$$\Delta S = \frac{Q}{T} = \frac{L}{T}$$

$L$ — 相变潜热；$T$ — 相变温度（等温等压可逆相变）

§5 熵增加原理 p228-234

★ 熵增加原理
$$\Delta S_{\text{孤立}} \geq 0$$

孤立系统的熵永不减少：可逆过程 $\Delta S = 0$，不可逆过程 $\Delta S > 0$。

热力学第二定律的数学表达：对于任意过程，系统和周围环境的总熵变满足：

$$\Delta S_{\text{总}} = \Delta S_{\text{系统}} + \Delta S_{\text{外界}} \geq 0$$

这是判断过程是否可逆的准则（熵判据）。

应用：自由膨胀的熵变 p231

理想气体向真空自由膨胀，体积从 $V_1$ 到 $V_2$：

$$\Delta S = nR\ln\frac{V_2}{V_1} > 0$$

自由膨胀是典型的不可逆过程，熵增加。

§6 热力学第二定律的统计意义 p234-240

玻尔兹曼熵公式 p236

$$S = k_B \ln \Omega$$

$\Omega$ — 宏观状态对应的微观状态数；$k_B$ — 玻尔兹曼常数

统计意义：

  • 孤立系统的自发过程总是从微观状态数少的宏观态（有序）向微观状态数多的宏观态（无序）过渡
  • 熵是系统无序程度的量度
  • 不可逆性的统计本质：逆向过程的概率极小（如 $N$ 个分子全回到原处的概率为 $1/2^N$）
  • 对一摩尔气体，全部分子返回容器一侧的概率几乎为零（$1/2^{6\times10^{23}}$）

自由膨胀的统计解释

一个分子在容器左右两侧各有 2 种等概率分布。$N$ 个分子共有 $2^N$ 种分布。

$$\Delta S = k_B \ln \frac{\Omega_2}{\Omega_1} = k_B \ln(2^N) = Nk_B\ln 2$$

与热力学结果 $nR\ln 2$ 一致（因 $Nk_B = nR$）。

§7 热力学第三定律 p240-244

★ 热力学第三定律（能斯特热定理）

不可能通过有限步骤使任何物体冷却到绝对零度（0 K）。

等价表述：当 $T \to 0$ K 时，任何纯物质的完美晶体的熵趋于零。

$$\lim_{T\to 0} S(T) = 0$$

第三定律的推论

  • 绝对零度不可达：无论怎样冷却，无法通过有限次操作到达 0 K
  • 低温热容：$T \to 0$ 时，$C_V, C_P \to 0$（德拜 $T^3$ 律）
  • 提供了熵的绝对基准（$S(0)=0$）
$$S(T) = \int_0^T \frac{C(T')}{T'}\,dT'$$

§8 熵与能量退降 p244-250

不可逆过程导致能量"退降"：虽然总能量守恒，但可用能（可转化为功的部分）减少。熵的增加反映了能量的退降。

$$E_{\text{退降}} = T_0 \Delta S$$

$T_0$ — 环境温度；$\Delta S$ — 不可逆过程引起的熵增
