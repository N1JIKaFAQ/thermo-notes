第五章 热力学第一定律

李椿·章立源·钱尚武《热学》 — 期末补热学笔记

概述 p144

热力学第一定律是能量守恒定律在热力学过程中的具体体现。本章在准静态过程和功的概念基础上，建立热力学第一定律，并应用于理想气体的各种典型过程（等容、等压、等温、绝热、多方），最后讨论循环过程和卡诺循环。

§1 热力学过程 p144-148

一、准静态过程 p144-146

  • 热力学过程：系统状态随时间变化的过程
  • 非静态过程：变化太快，系统经历一系列非平衡态
  • 准静态过程：过程无限缓慢，每一时刻系统都近似处于平衡态
  • 弛豫时间：从旧平衡破坏到新平衡建立所需的时间

准静态过程的条件：过程进行时间 $\gg$ 弛豫时间，且无摩擦。

二、p-V 图 p146-148

  • 以 $p$ 为纵坐标、$V$ 为横坐标的图
  • 每一点代表一个平衡态
  • 每一条连续曲线代表一个准静态过程
  • 非平衡态不能在 p-V 图上表示

§2 功 p148-152

在准静态过程中，外界对系统所做的元功：

★ 准静态过程的功 (5.2)
$$dA = -p\,dV$$ $$A = -\int_{V_1}^{V_2} p\,dV \tag{5.3}$$
  • 系统压缩（$dV<0$）：$dA>0$，外界对系统做正功
  • 系统膨胀（$dV>0$）：$dA<0$，系统对外界做正功
  • 系统对外做功：$dA' = p\,dV = -dA$

功是过程量：其数值不仅取决于初末状态，还取决于路径。在 p-V 图上，功等于曲线下的面积（带符号）。

功不是状态函数，$dA$ 不是全微分。

§3 热量与热容 p153-158

一、热量 p153-154

热量是系统与外界间由于温度差而传递的能量，是过程量。焦耳实验（重物下落搅水、电流加热等）证明了热与功的等价性：

$$\text{热功当量} \quad 1\,\text{cal} = 4.184\,\text{J}$$

二、热容 p154-158

名称定义式对理想气体
热容 $C$$C = \dfrac{dQ}{dT}$—
定容摩尔热容 $C_V$$C_V = \left(\dfrac{\partial U}{\partial T}\right)_V$$C_V = \frac12(t+r+2s)R$
定压摩尔热容 $C_P$$C_P = \left(\dfrac{\partial H}{\partial T}\right)_p$$C_P = C_V + R$
比热比 $\gamma$$\gamma = C_P/C_V$单原子 1.67, 双原子 1.40
★ 迈耶公式 (5.8)
$$C_P - C_V = R$$

对理想气体成立，源于 $H = U + pV$ 关系。

§4 热力学第一定律 p159-164

★ 热力学第一定律
$$\Delta U = Q - A \quad\text{(系统对外做功为正)}$$ $$\Delta U = Q + W \quad\text{(外界对系统做功为正)}$$

微分形式：$dU = \delta Q - \delta A$

其中 $\Delta U$ 为内能变化（状态函数，只取决于初末态），$Q$ 为吸收热量，$A$ 为系统对外做功。

内能是状态函数：对理想气体，$U$ 仅与温度有关，$U = \frac12(t+r+2s)RT$。

对任何循环过程，$\oint dU = 0$，所以 $Q_{\text{循环}} = A_{\text{循环}}$。

§5 焓 p164-167

★ 焓的定义
$$H = U + pV$$

焓是状态函数。在等压过程中（$dp=0$）：

$$\Delta H = Q_p$$

即等压过程中系统吸收的热量等于焓的增加。

焓的微分形式

$$dH = dU + p\,dV + V\,dp = \delta Q + V\,dp$$

等压时 $dp=0$ → $dH = \delta Q_p$

§6 热力学第一定律对理想气体的应用 p168-180

过程条件$A$（系统做功）$Q$$\Delta U$过程方程
等容 $V=\text{const}$ $0$ $nC_V\Delta T$ $nC_V\Delta T$ $\dfrac{p}{T}=\text{const}$
等压 $p=\text{const}$ $p\Delta V = nR\Delta T$ $nC_P\Delta T$ $nC_V\Delta T$ $\dfrac{V}{T}=\text{const}$
等温 $T=\text{const}$ $nRT\ln\dfrac{V_2}{V_1}$ $A$ $0$ $pV=\text{const}$
绝热 $Q=0$ $-\Delta U = nC_V(T_1-T_2)$ $0$ $-A$ $pV^\gamma=\text{const}$

一、等容过程 p168-170

$dV=0 \Rightarrow A=0$，热量全部用于增加内能。

$$Q_V = \Delta U = nC_V(T_2 - T_1)$$

二、等压过程 p170-172

$p=\text{const}$，系统做功 $A = p(V_2 - V_1)$。

$$A = nR(T_2 - T_1)$$ $$Q_P = nC_P(T_2 - T_1)$$ $$\Delta U = nC_V(T_2 - T_1)$$

三、等温过程 p172-175

$T=\text{const}$，理想气体内能不变（$\Delta U=0$）。吸收的热量全部用于对外做功。

$$A = \int_{V_1}^{V_2} p\,dV = nRT\int_{V_1}^{V_2}\frac{dV}{V} = nRT\ln\frac{V_2}{V_1}$$ $$Q = A = nRT\ln\frac{V_2}{V_1} = nRT\ln\frac{p_1}{p_2}$$

四、绝热过程 p175-178

$Q=0$，系统对外做功完全来自内能减少。

★ 绝热过程方程（泊松公式）
$$pV^\gamma = \text{const} \tag{5.26}$$ $$TV^{\gamma-1} = \text{const} \tag{5.27}$$ $$p^{\gamma-1}T^{-\gamma} = \text{const}$$
绝热过程的功
$$A = -\Delta U = nC_V(T_1 - T_2)$$ $$A = \frac{p_1V_1 - p_2V_2}{\gamma-1} = \frac{nR(T_1 - T_2)}{\gamma-1}$$

绝热线与等温线的比较：在 p-V 图上，绝热线比等温线更陡。因为 $\gamma > 1$，绝热膨胀时压强下降更快（温度同时下降）。

五、多方过程 p178-180

实际过程通常介于等温与绝热之间，可用多方过程描述：

$$pV^n = \text{const} \quad (1 < n < \gamma)$$

$n$ 为多方指数。$n=0$ 等压、$n=1$ 等温、$n=\gamma$ 绝热、$n\to\infty$ 等容。

§7 循环过程和卡诺循环 p180-191

一、循环过程 p180-183

系统经过一系列变化后回到初态的过程。在 p-V 图上为闭合曲线。

$$\oint dU = 0 \quad\Rightarrow\quad A_{\text{净}} = Q_{\text{净}} = Q_1 - Q_2$$

$Q_1$ — 系统吸收的总热量；$Q_2$ — 系统放出的总热量（绝对值）。

热机效率

$$\eta = \frac{A_{\text{净}}}{Q_1} = 1 - \frac{Q_2}{Q_1} \tag{5.31}$$

二、卡诺循环 p183-191

卡诺循环由两个等温过程和两个绝热过程组成，是效率最高的热机循环。

★ 卡诺热机效率
$$\eta_C = 1 - \frac{T_2}{T_1} \tag{5.34}$$

$T_1$ — 高温热源温度；$T_2$ — 低温热源温度

卡诺循环的四个过程

  1. 等温膨胀（$T_1$）：与高温热源接触，吸热 $Q_1 = nRT_1\ln(V_2/V_1)$
  2. 绝热膨胀：温度降至 $T_2$
  3. 等温压缩（$T_2$）：与低温热源接触，放热 $Q_2 = nRT_2\ln(V_3/V_4)$
  4. 绝热压缩：温度升至 $T_1$，回到初态

可证：$\dfrac{Q_1}{T_1} = \dfrac{Q_2}{T_2}$（卡诺定理的重要基础）

卡诺定理：

  1. 在相同的高温热源和低温热源之间工作的一切可逆热机，其效率都相等，与工作物质无关
  2. 在相同的高温热源和低温热源之间工作的一切不可逆热机，其效率都小于可逆热机的效率
