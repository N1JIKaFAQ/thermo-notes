第一章 温度

李椿·章立源·钱尚武《热学》 — 期末补热学笔记

§1 平衡态 状态参量

平衡态：在不受外界影响的条件下，系统宏观性质不随时间变化的状态。
本质是热动平衡——分子永不停息运动，但平均效果不变。平衡态是理想概念。

状态参量（四类）

类型举例符号
几何参量体积$V$
力学参量压强$p$
化学参量摩尔数、浓度$\nu$
电磁参量电场强度、磁化强度$E, M$

一定质量的化学纯气体，用 $p$, $V$ 即可完全描述。

§2 温度

热力学第零定律

若 A 与 B 热平衡，A 与 C 热平衡 → B 与 C 也热平衡。

→ 温度的定义：决定系统是否与其他系统处于热平衡的宏观性质。

互为热平衡的系统 温度相同。

（在附录 1-1 中给出了严格的数学推导）

温标三要素

  1. 测温物质（水银、铂电阻、气体……）
  2. 测温属性（液柱长度、电阻值、压强……）
  3. 固定点 + 刻度规定

标准固定点：水的三相点

水的三相点 = 纯冰 + 纯水 + 水蒸气 平衡共存

温度严格规定为 273.16 K

（附录 1-2 介绍了实现三相点的实验装置）

四种温标

温标符号单位定义
理想气体温标$T$K$T = 273.16 \cdot \dfrac{p}{p_{\text{tr}}}$（取$p_{\text{tr}}\to0$极限）
热力学温标$T$K最基础温标，与气体温标在适用范围内一致
摄氏温标$t$°C$t = T - 273.15$
华氏温标$t_F$°F$t_F = 32 + \dfrac{9}{5}t$
★ 理想气体温标（定容气体温度计）
$$T = \lim_{p_{\text{tr}}\to 0} 273.16\,\text{K} \cdot \frac{p}{p_{\text{tr}}}$$
★ 摄氏 ↔ 开尔文
$$t(^\circ\text{C}) = T(\text{K}) - 273.15$$

理想气体温标的实现

用四种不同气体做实验，测汽点温度 $T(p_s)$ 对 $p_{\text{tr}}$ 作图，外推到 $p_{\text{tr}}\to0$，不同气体的温标趋于同一极限值 373.15 K。

这个极限温标就是 理想气体温标，与热力学温标一致。

气体温度计

  • 定容：V 不变，p 随 T 变 → $T = 273.16 \cdot p/p_{\text{tr}}$
  • 定压：p 不变，V 随 T 变 → $T = 273.16 \cdot V/V_{\text{tr}}$

两者的极限（$p\to0$）给出相同的温标。

理想气体温标局限性：最低 1 K（氦液化点），最高约 1000°C。极低温不适用。

§3 气体的状态方程

3.1 理想气体状态方程

由三条实验定律导出：

  • 玻意耳定律（Boyle）：$T$不变 → $pV = C$
  • 盖·吕萨克定律：$p$不变 → $V = V_0(1 + \alpha_v t)$，$\alpha_v = 1/273.15$
  • 查理定律（Charles）：$V$不变 → $p = p_0(1 + \alpha_p t)$，$\alpha_p = 1/273.15$
★★★ 核心公式 — 理想气体状态方程
$$pV = \nu RT = \frac{M}{\mu}RT$$
符号含义常用单位
$p$压强atm 或 Pa
$V$体积L 或 m³
$\nu$摩尔数 $\nu = M/\mu$mol
$M$气体质量kg 或 g
$\mu$摩尔质量kg/mol（数值上 = 分子量）
$R$普适气体常数见下方
$T$热力学温度K

R 的取值（按单位选）

单位制$R$ 数值
国际单位制（p: Pa, V: m³）$R = 8.314\ \text{J}\cdot\text{mol}^{-1}\cdot\text{K}^{-1}$
常用（p: atm, V: L）$R = 0.08206\ \text{atm}\cdot\text{L}\cdot\text{mol}^{-1}\cdot\text{K}^{-1}$
书上例题近似$R = 8.21 \times 10^{-2}\ \text{atm}\cdot\text{L}\cdot\text{mol}^{-1}\cdot\text{K}^{-1}$

阿伏伽德罗定律

同温同压下，等摩尔数的任何气体体积相同。

标准状况（STP: $0^\circ\text{C}$, 1 atm）下，1 mol 气体体积 = 22.4 L

★ 密度公式
$$\rho = \frac{M}{V} = \frac{p\mu}{RT}$$

3.2 混合理想气体

道耳顿分压定律

$$p_{\text{总}} = p_1 + p_2 + \cdots + p_n$$

某组分分压 = 该组分单独存在（同 T 同 V）时的压强。

混合气体状态方程：$pV = \nu RT$，其中 $\nu = \nu_1 + \nu_2 + \cdots$

★ 平均摩尔质量
$$\mu_{\text{avg}} = \frac{M_{\text{总}}}{\nu_{\text{总}}}$$

例：空气（N₂ 76.9%, O₂ 23.1%）→ $\mu_{\text{avg}} \approx 28.9\ \text{g/mol}$

3.3 非理想气体 — 范德瓦耳斯方程

★ 对 1 摩尔气体
$$\left(p + \frac{a}{v^2}\right)(v - b) = RT$$
参数含义典型值（CO₂）
$a$修正分子间吸引力3.592 atm·L²·mol⁻²
$b$修正分子本身体积0.04267 L·mol⁻¹

a 和 b 的数值见教材表 1-2，不同气体不同。

例题 5 的解法示范

将方程改写为：

$$p = \frac{RT}{v - b} - \frac{a}{v^2}$$

代入 $T, v, a, b$ 即可求出压强 $p$。

对任意质量气体（质量 M，总体积 V）
$$\left[ p + \frac{M^2}{\mu^2}\frac{a}{V^2} \right] \left[ V - \frac{M}{\mu}b \right] = \frac{M}{\mu}RT$$

3.4 昂尼斯方程（维里展开）

$$pv = A + Bp + Cp^2 + Dp^3 + \cdots$$

$A = RT$，$B, C, D\cdots$ 叫 维里系数，是温度的函数。

也可以写成 $pv = A + B_v/v + C_v/v^2 + D_v/v^3 + \cdots$，两种形式可互相转换。

范德瓦耳斯方程也可以展开成维里形式：

$$pv = RT + \frac{(RTb - a)}{v} + \frac{RTb^2}{v^2} + \cdots$$
